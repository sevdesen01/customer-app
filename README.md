# Customer Management System
This is a minimalist and responsive Customer Management System web application built with React. Designed with a modern, vibrant gradient background, it provides a clean and user-friendly interface for managing customer lists efficiently.

Key Features:

Add Customers: Easily add new customer names to the list via an intuitive input field.

Dynamic List: View the added customers dynamically with a modern UI card layout.

Delete Functionality: Remove customers from the system instantly using the quick-delete action button.

Responsive & Clean UI: A simple, eye-catching design optimized for a smooth user experience.
